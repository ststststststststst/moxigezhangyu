洛谷题解：SP7169 EGYPIZZA - Pizza

前言

这是本蒟蒻的第一篇题解，请多多指教！

题意：

输入 n 个分数，求向上取整所有分数的和并加 1 后的值。

题目分析：

先通分，单位统一，这样好算一点。然后把每个人吃的块数（分子）相加（别忘 Abotrika！），最后再除以 4 在向上取整，就是最后的答案。

代码:

#include<bits/stdc++.h>
using namespace std;
int n;
string s;//储存分数
int main(){
    cin>>n;
    int ans=0;
    while(n--){
        cin>>s;
        if(s=="1/4")ans+=1;// 1/4
        else if(s=="1/2")ans+=2;//1/2||2/4
        else ans+=3;//3/4
    }
    ans+=1;//别忘了！
    printf("%d\n",ceil(ans/4)+1);//输出
    return 0;//养成好习惯！
}
